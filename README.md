# 🎓 Student CRUD App (React + JSON Server)

A simple **CRUD (Create, Read, Update, Delete)** application built using **React (Vite)** and **JSON Server**.
This project helps manage student data like **ID, Name, and Email** with a clean UI.

---

## 🚀 Features

* ➕ Add new student
* 📋 Display all students
* ✏️ Edit student details
* ❌ Delete student
* 🔄 Real-time updates using JSON Server API

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🌐 REST API (JSON Server)
* 🎨 Bootstrap (for styling)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/student-crud-app.git
cd student-crud-app
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Run JSON Server

```
npx json-server --watch db.json --port 3000
```

### 4️⃣ Run React App

```
npm run dev
```
